# SaneSpace

> **Accessible, affordable, and culturally-aware mental wellness support.**

SaneSpace is an AI-powered mental wellness platform designed to make emotional support more accessible for Nigerians and underserved communities. By combining AI, mood tracking, and culturally relevant resources, SaneSpace helps users better understand and manage their emotional well-being in a safe and judgment-free environment.

---

##  Problem

Millions of people struggle to access quality mental health support because of:

* High therapy costs
* Cultural stigma surrounding mental health
* Limited availability of professionals
* Lack of support for Nigerian English and Pidgin
* Inadequate early intervention tools

SaneSpace aims to bridge this gap by providing accessible and culturally relevant mental wellness support.

---

##  Features

*  **AI Companion** for empathetic conversations

##  Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/sanespace.git
cd sanespace
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
